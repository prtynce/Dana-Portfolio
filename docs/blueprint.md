# **App Name**: Dana: Evolved Portfolio

## Core Features:

- Interactive Portfolio Showcase: Display projects, achievements, and skills with detailed views, filtering, hover effects, and carousels for an engaging experience.
- Dynamic Personal Profile: An immersive hero section with animated gradients, floating abstract shapes, typing text for subtitle, and an 'About Me' with interactive profile card and skill visualizations.
- Smart Theme & Interaction System: Implement a user-toggleable light/dark mode with smooth transitions, auto-mode based on system preference, and core micro-interactions like cursor glow, button ripple, and card hover effects.
- Flexible Content Placeholders: A simulated 'Edit Mode' (for future CMS integration) allowing client-side demonstration of editable links, tags, and drag-and-drop image upload placeholders for portfolio items.
- AI-Powered Experiment Ideas: In the 'Experiments/Lab' section, leverage an AI tool to generate or suggest concise, trending descriptions for emerging tech concepts in AI and Robotics.
- Enhanced Navigation & Accessibility: A sticky, glassmorphism navigation bar with an active section indicator, scroll progress bar, and an animated mobile hamburger menu for intuitive browsing.
- Integrated Contact & Social Links: A modern contact form with floating labels and input glow on focus, alongside an email copy button and interactive social media icons with subtle animations.

## Style Guidelines:

- Primary Color: An elegant, vibrant rose pink (#D169AF), balancing the 'feminine' and 'futuristic' aesthetic without being overly bold. This serves as a highlight and accent for interactive elements.
- Background Color: A very light, almost ethereal blush pink (#F8EFF6). Its high lightness and low saturation ensure high readability and a clean base for the light mode's layered gradients, while subtly reflecting the primary pink hue.
- Accent Color: A soft lavender (#CC88E5) to provide a harmonious, analogous contrast to the primary pink, adding depth and complementing the 'futuristic glow' without overwhelming the overall softness. This color will be used for subtle highlights and secondary interactive elements.
- Headlines Font: 'Space Grotesk', a proportional sans-serif with a techy, scientific feel, for bold, elegant headings that align with the 'slightly futuristic' and professional aesthetic. 
- Body Font: 'Inter', a grotesque-style sans-serif, offering an ultra-readable, clean, and modern look for all body text, ensuring high readability and a sleek appearance. 
- Use minimalist, vector-based icons with subtle glowing outlines for interactive elements and futuristic flair, maintaining readability against glassmorphism effects. Tech-related icons (AI, robotics, circuits) should align with this style.
- Utilize a modular card-based layout, prominent glassmorphism and soft UI elements for cards and navigation. The hero section should employ a split-screen or centered approach with layered, animated backgrounds. Project sections feature flexible grids and carousel displays, with a distinctive vertical timeline for achievements.
- Implement extensive, smooth animations including a soft pink cursor glow, subtle button ripple effects on click, cards lifting and casting shadows on hover, animated text reveals (fade + slide) for headings, scroll-triggered fade/slide/scale transitions, and fluid page navigation with parallax scrolling effects for an immersive SPA feel.
